# Week #005 + Week #006
## 在 express 上實作 sequelize 與 MySQL 的串接
### 專案啟動步驟
1. 下載並安裝 MySQL 伺服器在本地電腦，建議版本為 v8.0
2. 啟動本地 MySQL 伺服器
3. 安裝專案裡的模組
	```node
	npm install
	```
4. 啟動 Node 伺服器  
	```node
	npm run dev
	```
### 開發環境依賴
- MySQL v8.0
- Node.js v14.21.1
